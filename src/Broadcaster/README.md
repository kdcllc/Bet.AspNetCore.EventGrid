# Broadcaster

This is a simple test app based on DotNetCore 3.0 Worker template.

This application tests submission of Event Grid Topics based on:

- Event Grid Schema
- Cloud Event Schema

In order to test the app provide the following settings for Azure Event Grid Topics:

```json
  // vault value
  "TopicClientEventGrid": {
    "Endpoint": "",
    "Key": ""
  },

  // vault value
  "TopicClientCloudEvent": {
    "Endpoint": "",
    "Key": ""
  }
```
